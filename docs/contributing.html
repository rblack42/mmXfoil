Contributing¶

If you would like to help with this project, please follow these instructions:

  1. Create a “fork” of the main project on GitHub. This will create a copy of the code in your account.

  2. Clone the copy to your workstation

  3. Make any changes you like, pushing them to your account as needed.

  4. Issue a “Pull Request” that will notify me of proposed changes. I will review then and merge them into the project if they look useful.

Requesting a Feature¶

If you think of a feature you think would make this tool more useful to model builders, please generate a “Feature request”. This can be done from the Issues menu item on the project age.
